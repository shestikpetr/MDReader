# MD Reader

Легковесное десктопное приложение для просмотра Markdown-файлов написанное на Kotlin с помощью Compose Multiplatform.

## Возможности

- **Рендеринг Markdown** — заголовки, параграфы, жирный, курсив, зачёркнутый текст, блоки кода с подсветкой синтаксиса, цитаты, маркированные/нумерованные списки, таблицы, изображения, ссылки, горизонтальные разделители
- **Оглавление (TOC)** — автоматически генерируемая боковая панель с кликабельными заголовками (H1–H6) для быстрой навигации
- **Изменяемые панели** — перетаскиваемые разделители между панелями TOC и контента
- **Масштабирование** — `Ctrl+=`/`Ctrl+-` для увеличения/уменьшения, `Ctrl+0` для сброса, `Ctrl+Scroll` для зума колёсиком мыши, двойной клик по проценту для сброса
- **Тёмная/Светлая тема** — переключение в панели инструментов
- **Авто-обновление** — отслеживает изменения файла и обновляет содержимое автоматически
- **Drag & Drop** — перетащите `.md` файл в окно для открытия
- **Диалог открытия** — `Ctrl+O` для выбора файла через системный диалог
- **Статус-бар** — количество слов, символов и примерное время чтения
- **Копирование кода** — кнопка копирования появляется при наведении на блок кода
- **GFM** — поддержка таблиц и зачёркнутого текста GitHub Flavored Markdown через расширения Commonmark

## Стек технологий

| Компонент | Библиотека |
|-----------|------------|
| Язык | Kotlin 2.3.0 |
| UI | Compose Multiplatform (Desktop) 1.7.3 |
| Дизайн | Material 3 |
| Парсер Markdown | [Commonmark](https://github.com/commonmark/commonmark-java) 0.24.0 |
| Иконки | Material Icons Extended |


## Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+O` | Открыть файл |
| `Ctrl+=` | Увеличить масштаб |
| `Ctrl+-` | Уменьшить масштаб |
| `Ctrl+0` | Сбросить масштаб |
| `Ctrl+Scroll` | Масштабирование колёсиком |

## Структура проекта

```
src/main/kotlin/
├── Main.kt                          # Точка входа, окно, хоткеи, drag & drop
├── file/
│   └── FileChooser.kt               # Системный диалог выбора файла
├── markdown/
│   ├── MdParser.kt                  # Настройка парсера Commonmark
│   ├── MarkdownViewer.kt            # Рендерер на основе LazyColumn
│   ├── HeadingEntry.kt              # Извлечение заголовков для TOC
│   ├── TextScale.kt                 # CompositionLocal для масштаба
│   ├── highlight/
│   │   └── SyntaxHighlighter.kt     # Подсветка синтаксиса в блоках кода
│   ├── blocks/
│   │   ├── MdBlockDispatch.kt       # Диспетчер блочных узлов
│   │   ├── MdHeadingBlock.kt        # Рендеринг заголовков
│   │   ├── MdParagraphBlock.kt      # Рендеринг параграфов
│   │   ├── MdCodeBlock.kt           # Блоки кода (fenced и indented)
│   │   ├── MdQuoteBlock.kt          # Цитаты
│   │   ├── MdListBlocks.kt          # Маркированные и нумерованные списки
│   │   └── MdTableBlock.kt          # GFM-таблицы
│   └── inline/
│       └── MdInlineRenderer.kt      # Инлайн-разметка (жирный, курсив и т.д.)
└── ui/
    ├── AppToolbar.kt                # Верхняя панель инструментов
    ├── StatusBar.kt                 # Нижняя статус-панель
    ├── ResizableLayout.kt           # Раскладка с изменяемыми панелями
    ├── SidePanel.kt                 # Боковая панель с оглавлением
    ├── DragHandle.kt                # Перетаскиваемый разделитель панелей
    └── WelcomeScreen.kt             # Заглушка при отсутствии файла
```
